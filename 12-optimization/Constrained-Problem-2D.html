def A(x):
    d1 = x[0]
    d2 = x[1]
    return d1*d2


def P(x):
    d1 = x[0]
    d2 = x[1]
    return 2*(d1+d2)

Constrained Optimization Problem¶

max A

st P <= 20

or to use with minimize

min f = -A

st g = 20 - P >= 0
